# SEBINOR Computación - Sitio Web

Sitio web de SEBINOR Computación. Construido en HTML/CSS/JS puro.

## Estructura
- `index.html` — Página principal
- `pages/` — Páginas internas (outlet, equipos, monitores, etc.)
- `css/style.css` — Estilos globales
- `js/main.js` — JavaScript

## Para publicar en GitHub Pages
1. Subir todo a un repositorio de GitHub
2. Ir a Settings → Pages → Branch: main → Save
3. El sitio queda en: `https://tu-usuario.github.io/sebinor/`

## Para agregar/editar monitores
Editar el archivo `pages/monitores.html` y agregar nuevas tarjetas siguiendo el mismo formato.
